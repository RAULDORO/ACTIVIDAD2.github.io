Archivo plano

Los archivos planos son una coleccion de informacion que esta almacenada y es accedida de forma organizada en una base de datos. Se suele usar para almacenar informacion en una forma no estructurada.


Ejemplo sobre el archivo plano.

Para poner un ejemplo, cuando hace un pedido, alguien inserta esta informacion en la tabla de este cliente dentro de la base de datos. Otra persona puede poner informacion de su pedido en una tabla diferente. Cada una de estas tablas incluyen campos que los conectan entre si.

Archivo XML

es definido como un lenguaje marcado para documentos que contienen informacion estructurada que posee contenido (como palabras o imagenes) y alguna indicacion del papel que desempena.

Como abrir un Archivo XML

En muchas ocasiones bastara con abrirlo y editarlo con un editor de texto. O incluso es posible abrirlo y visualizarlo con la ayuda de un navegador web. Los pasos a seguir en Windows son bastante simples. Solo tenemos que situarnos con el raton encima del archivo XML, pulsar sobre el con el boton derecho y seleccionar Abrir con a partir del submenu contextual que aparece. Finalmente, escogemos la opcion Bloc de notas.

Archivo json

Los archivos tipo json (JavaScript Object Notation), es un formato el cual es implementado por Java. Hace parte de uno de los lenguajes mas importantes de la programacion y tiene como fin la creacion de paginas web como tambien en la generacion de multiples software. Este tipo de lenguaje es uno de los mas complicados de desarrollar, pero a su vez es implementado en los navegadores con mayor flujo como lo son Google y Yahoo.

Programas para abrir archivos JSON

Block de notas, Mozilla firefox, Atom, Notepad++, wordpad

Archivo parquet

es un formato de archivo en columnas que proporciona optimizaciones para acelerar las consultas. Es un formato de archivo mucho mas eficaz que el archivo .csv o JSON.
